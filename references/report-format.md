# IC Analysis 综合诊断报告格式

跑完 5 项分析后输出**综合诊断报告**。模板：

```
=== IC Analysis Report ===
Factor      : f_neg_max_ret_120
Period      : 2021-12-04 → 2024-12-03 (val)
Universe    : CSI1000 (动态)

[1] 双 IC 对照
    rank IC mean    : +0.038       rank_ic_ir   : 2.14
    pearson IC mean : +0.029       pearson_ic_ir: 1.87
    IC > 0 占比     : 62%
    诊断            : 量级稍失真，rank > pearson by 30%

[2] IC Decay
    H=1   ic=0.012  ir=1.05
    H=3   ic=0.024  ir=1.62
    H=5   ic=0.031  ir=1.89
    H=10  ic=0.036  ir=2.05
    H=20  ic=0.038  ir=2.14   ← 当前 horizon
    H=40  ic=0.029  ir=1.74
    诊断  : 中频信号（H=20 峰值），适合 H ∈ [10, 20]

[3] 子样本 IC（按市值 5 分位）
    Q1 (smallest) : ic=0.062  ir=2.85   ← 主导
    Q2            : ic=0.048  ir=2.21
    Q3            : ic=0.033  ir=1.78
    Q4            : ic=0.024  ir=1.32
    Q5 (largest)  : ic=0.011  ir=0.55
    诊断  : 小票主导，实盘容量受限于小盘流动性

[3b] 子样本 IC（按 vol_20 5 分位）
    Q1 (low vol)  : ic=0.022
    Q5 (high vol) : ic=0.054   ← 高波股 IC 翻倍
    诊断  : 偏 lottery 因子，吃高波风险溢价

[4] Top 10% Jaccard
    mean=0.91  std=0.03
    诊断  : 信号变化慢，换手低，符合中频持仓

[5] IC 时序
    cum_ic 形态     : 一路上扬，无明显失效段
    最近 60d 滚动 IC: +0.041 (健康)

=== 综合判读 ===
- 因子类型 : 中频 lottery 因子，主要在小盘高波股上有效
- 推荐 H   : 20（当前最优；H=10 IC 略低但换手降低 30%）
- 实盘风险 : 容量受限（小盘流动性），高波股集中（MDD 可能偏深）
- 后续动作 : 考虑加一个低波因子做对冲（H_action: combine_method 而非新因子）
```

## 必含元素

| 区块 | 内容 |
|---|---|
| **元信息** | Factor / Period / Universe |
| **§1 双 IC** | rank / pearson 对照，IC > 0 占比 |
| **§2 IC Decay** | 至少 5 个 horizon 的 IC + IC_IR |
| **§3 子样本** | 至少市值切分（必），波动 / 行业（推荐） |
| **§4 Jaccard** | 平均值 + 含义 |
| **§5 时序** | 形态描述 + 最近滚动 IC 状态 |
| **综合判读** | 因子类型 / 推荐 H / 实盘风险 / 后续动作 |

## 不要做的事

- ❌ 只给一段文字总结："因子表现不错"
- ❌ §3 不切，只说"整体 IC 0.04"
- ❌ §5 没看时序就说"稳定有效"
- ❌ 综合判读和具体数据脱节（"建议加新因子"但没说为什么）

## 输出长度参考

- 完整报告 ~30 ~ 50 行（含数据 + 诊断）
- 综合判读区 ~5 行（factor 类型 / 推荐 H / 风险 / 动作）
- 不要超过 80 行 —— 太长就把细节移到附图
